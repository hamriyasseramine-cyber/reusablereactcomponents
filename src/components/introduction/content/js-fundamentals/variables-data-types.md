# Variables & data types

JavaScript has two main ways to declare variables you'll use constantly in React: `let` (for values that can change) and `const` (for values that shouldn't be reassigned). In React, you'll use `const` most of the time, especially for components and functions.

```js
const name = "Yasser";
let count = 0;
count = count + 1;
```

The main data types you'll run into are strings, numbers, booleans, arrays, and objects — all of which show up constantly when working with component props and state.
