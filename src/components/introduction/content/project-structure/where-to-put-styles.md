# Where to put styles

There are a few common approaches for styling in React:

- **CSS files per component**, placed next to the component itself (e.g. `Navbar.jsx` + `Navbar.css`)
- **CSS Modules**, which scope styles automatically to avoid class name collisions
- **Inline styles**, useful for small, dynamic style changes based on props or state

For most projects, one CSS file per component next to it keeps things simple and easy to find.
