# Folder structure basics

A typical React project separates concerns into clear folders so you always know where to find things:

```
src/
  components/   -> reusable UI pieces
  pages/        -> full page components (one per route)
  assets/       -> images, icons, fonts
  hooks/        -> custom reusable hooks
```

Keeping this separation consistent as your project grows makes it much easier to navigate, even months later.
